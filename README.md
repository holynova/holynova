## 收藏夹淘金
- 这是一个很简单的 Chrome 插件。 它会在你打开新标签页时，从你的收藏夹里随机挑 3 条出来给你看。 点开算一次访问，你每次看到它也会记录一次展示。

- 你可以点“再翻三张”，继续随机看。 书签有新增或删除，它会自己更新。
